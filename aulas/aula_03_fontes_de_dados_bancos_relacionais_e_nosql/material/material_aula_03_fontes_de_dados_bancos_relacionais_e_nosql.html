Fontes de Dados: Bancos Relacionais e NoSQL

Material inicial para consolidar decisões de ingestão a partir de bancos relacionais e NoSQL na arquitetura de dados.

Tópicos-base

  • RDS, motores relacionais e extração incremental.
  • CDC para sincronização com o Data Lake.
  • DynamoDB e modelagem para alto throughput.

Exercícios de fixação

  1. Comparar um caso de uso ideal para RDS e outro para DynamoDB.
  2. Explicar benefícios e limitações do CDC.
  3. Propor uma partition key para um catálogo de eventos.
